# DentalHub — система управления стоматологической клиникой

**DentalHub** — это веб-приложение на Python/FastAPI для управления работой стоматологической клиники. Система позволяет управлять записями пациентов, врачами, услугами, специализациями и имеет встроенную админ-панель.

## Возможности

- Управление записями на приём (Appointments)
- Размещение информации о врачах стоматологах
- Каталог предоставляемых услуг с прайс-листом
- Управление специализациями врачей
- истема аутентификации пользователей
- Админ-панель на базе SQLAdmin
- Отзывчивый веб-интерфейс
- REST API для интеграции


Основные технологии:
- **FastAPI** — веб-фреймворк
- **SQLAlchemy** — ORM
- **Alembic** — миграции БД
- **Pydantic** — валидация данных
- **SQLAdmin** — админ-панель
- **Celery** — асинхронные задачи
